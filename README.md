# htm-rs
Hierarchical Temporal Memory, in Rust

### Features:
- Python Bindings
- SDR Data Structures
- Reusable Synapses Class
- Spatial Pooler Algorithm
- Simple Scalar Encoder/Decoder

### Installation

1) Fork this repository
2) `cargo build --release`
3) `ln -s target/release/libhtm_rs.so htm_rs.so`
